# ファイルを手動で追加する手順

claspが未追跡ファイルを自動的に追加しない場合、以下の手順でApps Scriptエディタから手動で追加できます。

## 手順

1. **Apps Scriptエディタを開く**
   - https://script.google.com/d/1vUyb3TrBm57302puLaxFaDFVofq3kmF9Lo5wNBdmcs3LUJjh-5O5ygiO/edit

2. **div.gsファイルを追加**
   - 左側の「+」ボタンをクリック
   - 「スクリプト」を選択
   - ファイル名を「div」に変更
   - `scripts/div.gs`の内容をコピー＆ペースト

3. **project.gsファイルを追加**
   - 左側の「+」ボタンをクリック
   - 「スクリプト」を選択
   - ファイル名を「project」に変更
   - `scripts/project.gs`の内容をコピー＆ペースト

4. **保存**
   - すべてのファイルを保存（Ctrl+S / Cmd+S）

5. **確認**
   - `clasp pull`を実行して、ファイルが追加されたか確認
